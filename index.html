Politique de confidentialité

Explorea respecte votre vie privée et s'engage à protéger vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations.

1. Données collectées

  • Informations personnelles : nom d'utilisateur, adresse e-mail, photo de profil.
  • Données d'utilisation : cartes capturées, niveau, expérience.
  • Données techniques : identifiants Firebase pour l'authentification.

2. Utilisation des données

Nous utilisons vos données pour :

  • Fournir les fonctionnalités de l'application (quiz, cartes, communauté).
  • Améliorer l'expérience utilisateur.
  • Assurer la sécurité et prévenir les abus.

3. Partage des données

Nous ne partageons pas vos données personnelles avec des tiers, sauf si requis par la loi.

4. Sécurité

Nous utilisons Firebase pour stocker vos données de manière sécurisée.

5. Vos droits

Vous pouvez demander la suppression de vos données en nous contactant à l'adresse suivante : explorea.help@gmail.com.

6. Modifications

Cette politique peut être mise à jour. Nous vous informerons des changements importants.

Contact

Pour toute question, contactez-nous à explorea.help@gmail.com.
